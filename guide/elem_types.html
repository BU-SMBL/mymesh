Element Types#

MyMesh infers the element type by the length of each element’s node connectivity. This allows for flexible handling of mixed element meshes, however there is an ambiguity between the 2D quadrilateral element and the 3D tetrahedral element. By default, MyMesh assumes 4-node elements are tetrahedra in the absence of clear indicator that they are quads. When relevant, many functions allow quads to be specifically designated.
